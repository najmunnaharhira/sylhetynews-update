# ⚡ Quick Deploy - Admin to Firebase Hosting

## 🚀 3 Simple Steps

### Step 1: Login to Firebase
```bash
firebase login
```
*(Opens browser - sign in with your Google account)*

### Step 2: Set Project
```bash
firebase use sylhetly-news
```

### Step 3: Deploy!
```bash
firebase deploy --only hosting
```

**Done!** 🎉 Your admin will be live at: `https://sylhetly-news.web.app`
